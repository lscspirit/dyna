registerStore(name, spec)

Register a Store Register the specification (implementation) for a Store. A Store will not be instantiated at this point. The Store instance will be created when the Flux flow starts. Once created, the Store instance will be extended with four event methods: emitChange(), addChangeListener(), removeChangeListener() and waitFor(), plus flux() and eventDispatcherToken(). The Store spec must include a $processEvent() method. It may also include a $initialize() method for initializing the Store.
Parameters:
Name Type Description
name string name of the Store
spec Object store specification
Source:
  • flux/stores.js, line 107
Throws:
if name or spec is not valid
Type
Error
Example
// Register a new Store named "todoStore"
dyna.registerStore("TodoStore", {
  $initialize : function() {
    this.todo_list = [];
  },

  $processEvent : function(event) {
    if (event.eventName() == 'todo_add') {
      this.addItem(event.payload());
    }
  },

  addItem : function(item) {
    this.todo_list.push(item);
    this.emitChange();
  },

  todoList : function() {
    return this.todo_list;
  }
});
